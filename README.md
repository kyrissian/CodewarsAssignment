## 📌 Project Overview

This repository contains Python solutions to selected [Codewars](https://www.codewars.com) kata challenges, completed as part of the SE Foundations Knowledge Check. Each solution has been tested and verified to pass all Codewars test cases.

---

## 🧩 Challenges Completed

### 1. Even or Odd
**Difficulty:** 8 kyu

Write a function that takes an integer and returns whether it is even or odd.

- If the number is even, return `"Even"`
- If the number is odd, return `"Odd"`

---

### 2. Convert a Number to a String
**Difficulty:** 8 kyu

Write a function that converts a given number (integer) into a string.

- Example: `123` → `"123"`
- Example: `-100` → `"-100"`

---

### 3. Remove String Spaces
**Difficulty:** 8 kyu

Write a function that removes all spaces from a given string and returns the result.

- Example: `"8 j 8"` → `"8j8"`

---

### 4. Vowel Count
**Difficulty:** 8 kyu

Write a function that returns the number of vowels (`a`, `e`, `i`, `o`, `u`) in a given string.

- Example: `"hello"` → `3`

## 🚀 How to Run & Test the Solutions

The solutions in this repository are written to be tested on [Codewars](https://www.codewars.com). Follow these steps to verify each solution:

1. Clone the repository and open `codewars_solutions.py` in your code editor:
```bash
git clone <your-repo-url>
cd codewars-solutions
```

2. Open `codewars_solutions.py` and find the solution you want to test.

3. Go to [Codewars.com](https://www.codewars.com) and navigate to the corresponding challenge:
   - [Even or Odd](https://www.codewars.com/kata/53da3dbb4a5168369a0000fe)
   - [Convert a Number to a String](https://www.codewars.com/kata/5265326f5fda8eb1160004c8)
   - [Remove String Spaces](https://www.codewars.com/kata/57eae20f5500ad98e50002c5)
   - [Vowel Count](https://www.codewars.com/kata/54ff3102c1bad923760001f3)

4. Copy the solution from `codewars_solutions.py` and paste it into the Codewars code editor for the corresponding challenge.

5. Click **"Test"** to run the sample tests, then **"Attempt"** to run the full test suite.

6. A green ✅ confirms the solution passes all test cases!

> **Note:** Codewars requires a free account to submit solutions. You can sign up at [codewars.com](https://www.codewars.com).
